 Weather Go Lite
===============================

<img src="https://i.imgur.com/sd2pFS4.png" width="100">

Introduction
---

Weather Go Lite 為 [Weather Go](https://github.com/Rannddyyy/WeatherGo) 的輕量化版本，捨去了路徑選擇的功能，僅留下氣象預測的圖表。
</br>
另外再額外新增自訂區域的功能。
